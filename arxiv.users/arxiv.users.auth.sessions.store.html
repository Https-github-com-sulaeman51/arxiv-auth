arxiv.users.auth.sessions.store module¶

Internal service API for the distributed session store.

Used to create, delete, and verify user and client session.

class arxiv.users.auth.sessions.store.SessionStore(host, port, db, secret, duration=7200)[source]¶

Bases: object

Manages a connection to Redis.

In reality, the StrictRedis instance is thread safe, and connections are attached at the time a command is executed. This class simply provides a container for configuration.
