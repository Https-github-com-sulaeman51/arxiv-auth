arxiv.users.domain.from_dict(cls, data)[source]¶

Generate a NamedTuple instance from a dict, with recursion.

This is the inverse of to_dict().

It’s easy to instantiate NamedTuples from dicts, but it’s not so easy if a NamedTuple field is typed with another NamedTuple. This function instantiates a NamedTuple class cls with the data in dict data, and also instantiates any referenced NamedTuple classes expected by cls’s fields with the appropriate data in data.

Parameters:
  • cls (type) – Any NamedTuple class.
  • data (dict) – Data with which to instantiate cls and its children.
Returns:

An instance of cls.
Return type:

NamedTuple
Return type:

Any
